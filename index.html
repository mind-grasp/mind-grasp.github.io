MIND-Grasp

Multiview Imagination-guided Next-best-action Discovery for Occlusion-Aware Grasping
Anonymous Submission for CoRL 2026

Abstract

Occlusion poses a significant challenge to robotic grasping in cluttered scenes, especially when the target object is fully or severely occluded. In such scenarios, the robot must not only infer the hidden target geometry, but also actively discover observable regions and ensure that the grasped object can be safely withdrawn from clutter. We propose MIND-Grasp, a human-like occlusion-aware grasping framework that combines amodal 3D target imagination, target-centric occlusion cloud reasoning, multiview Next-Best-Action discovery, and retraction-aware grasp selection. Given a language instruction and RGB-D observations, MIND-Grasp constructs an amodal target hypothesis and progressively updates the scene through active observation. Once the target becomes sufficiently observable, the system verifies grasp candidates by treating the gripper and target as a composite body and checking whether the object can be safely retracted. If no safe retraction path exists, MIND-Grasp identifies and removes the most critical occluder, enabling robust target retrieval in challenging occluded tabletop scenes.

Method Overview

Amodal Target Imagination

MIND-Grasp first builds an amodal 3D hypothesis of the target from language and partial RGB-D observations, providing a geometric prior for occlusion reasoning.

Occlusion-Cloud-Guided Discovery

The system constructs a target-centric occlusion cloud and samples observable corridors to actively reduce target uncertainty through multiview exploration.

Qualitative Results

Occlusion-Aware Grasping Analysis

Why Active Discovery?

Severe occlusion makes single-view grasp prediction unreliable. Active discovery progressively recovers observable target regions and reduces ambiguity before grasp execution.

Why Retraction-Aware Grasping?

A grasp is not sufficient if the target cannot be withdrawn. MIND-Grasp explicitly checks whether the gripper-target composite body can safely retreat from clutter.
